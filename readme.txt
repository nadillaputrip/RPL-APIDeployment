Data yang akan ditampilkan adalah data yang tersimpan dalam file menu.json.
Kode yang dijalankan adalah kode dalam file main.py

Untuk menggunakannya, berikut adalah langkah yang dapat dilakukan:
1. Mengaktifkan environment dengan command pipenv shell
2. Menuliskan command uvicorn main:app --reload
3. Untuk dapat mengakses endpoint post, update, dan delete, anda dapat merujuk ke link program yang ditambah "/docs"
